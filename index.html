Introduction

@TODO : put a nice image in background Lidar point cloud are becoming more and more available. Devices are easy to get, not too expensive, and provide very accurate 3D points. Yet using those Billions of points is a challenge. pgPointCLoud is an open source PostgreSQL extension for storing point cloud (LIDAR) data and use it with PostGIS. It is very easy to use, robust and efficient. By sotring Lidar point in a PostgreSQL database, pgPointCloud ease many problems and allows a good integration with other geo-spatial data (vector,raster) into the common PostGIS framework.
